对含有多个关键字的结构进行排序，比如　vector<pair<int,int> >，　对于pair中的first按照递增排序，对于first相同的按照second递增排序

class cmp
{
 public:
  bool operator()(pair<int,int>p1,pair<int,int>p2)
  {
   return (p1.first == p2.first)?p1.second<p2.second:p1.first<p2.first;
  }
};
